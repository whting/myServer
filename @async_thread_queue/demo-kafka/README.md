# 前置条件
- zookeeper服务启动
> zkServer
- kafka服务启动
> .\bin\windows\kafka-server-start.bat .\config\server.properties

# 应用
- 创建主题并发送消息
> ProducerDemo-main

- 消息中间件
> [-kafka>zookeeper>kafka-] 

- 参与主题,接受消息
> ConsumerDemo-main
